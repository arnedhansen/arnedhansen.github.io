Research
AOC
MATLAB 88% R 6% Python 5%
Full analysis pipeline for the Alpha Oculomotor Control study. Sternberg and N-back working memory tasks with simultaneous EEG and eye-tracking to characterize alpha-band neural signatures of oculomotor control. Published as a Registered Report in Psychophysiology.
AOI
MATLAB Python R
Extension of the AOC pipeline with a restructured hybrid MATLAB/Python/R architecture, source-level statistics, and multiverse analysis.
GCP
MATLAB 97% R 3%
Analysis pipeline for the Gamma Concentric Protocol. Dynamic concentric grating fixation task with simultaneous EEG and eye-tracking to investigate gamma-band neural signatures of oculomotor control.
CVA
MATLAB 100%
EEG preprocessing and analysis pipeline for a CSF Volume Attenuation study, including event logging and source-level statistics.
IAB
MATLAB 100%
Inattentional blindness paradigm with EEG preprocessing and analysis, developed for a Matura (Swiss high school) outreach project.
DALAS
R MATLAB
Specification curve analysis pipeline for EEG channel-level and LCMV source-level (DLPFC, frontal) models comparing healthy and diagnosed participant groups across auditory/visual conditions.
