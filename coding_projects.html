Research
AOC: Alpha Oculomotor Control
MATLAB 88% R 6% Python 5%
Full analysis pipeline for the Alpha Oculomotor Control study. Sternberg and N-back working memory tasks with simultaneous EEG and eye-tracking to characterize alpha-band neural signatures of oculomotor control. Published as a Registered Report in Psychophysiology.
AOI: Alpha Oculomotor Inflation
Python 68% MATLAB 32%
Follow-up to the AOC study investigating whether alpha-band effects co-varying with eye movements are inflated by oculomotor artifacts. Extends the AOC pipeline with a restructured hybrid MATLAB/Python/R architecture, source-level beamforming, and multiverse analysis.
GCP: Gamma Contrast Perception
MATLAB 97% R 3%
Full analysis pipeline for the Gamma Contrast Perception study. Dynamic concentric grating fixation task with simultaneous EEG and eye-tracking to characterize gamma-band neural responses to visual contrast and their relationship to oculomotor activity.
CVA: CSF Volume Attenuation
MATLAB 100%
Analysis pipeline examining how individual differences in cerebrospinal fluid (CSF) volume attenuate scalp-recorded EEG signals. Includes preprocessing, source-level statistics, and event logging.
IAB: Inattentional Blindness
MATLAB 100%
EEG paradigm and analysis pipeline for an inattentional blindness experiment. Developed as a public outreach project for Swiss Matura (high school) students to demonstrate neuroscientific methods.
DALAS: Depression Alpha Asymmetry
R 100%
Specification curve analysis examining frontal alpha asymmetry as a neural marker of depression. EEG channel-level and LCMV beamformer source-level models (DLPFC, frontal) comparing healthy controls and clinically diagnosed participants across auditory and visual conditions.

ADA: Automated Data Analysis
Python 100%
Desktop GUI application for general-purpose exploratory data analysis and statistical testing. Supports CSV/Excel import, outlier detection (IQR, Z-score, MAD), t-tests, ANOVAs, and repeated-measures ANOVA with assumption checks and publication-quality raincloud plots. Work in progress.
