Open Science

My empirical work follows a pre-registered, open-science workflow throughout. Where possible, studies are submitted as registered reports, committing to hypotheses, methods, and analysis plans before data collection begins. All analysis code is version-controlled and published on GitHub, structured so that a reader can move from raw data to reported results without undocumented steps. Datasets are shared alongside code where regulations permit.

Multiverse analysis is used as a robustness tool, systematically varying preprocessing and analytical choices to test whether conclusions depend on theoretically arbitrary decisions, which makes the fragility or stability of a finding explicit.

Modulations of posterior alpha power during working memory co-vary with task-dependent eye movement patterns
Preregistered+ Open Data Open Materials Open Code
Hansen, A.D., Strzelczyk, D., Troendle, M., Langer, N. & Popov, T. (2024)
[Psychophysiology — Registered Report (Stage 1 In-Principle Acceptance)] · [Code]
This study asked whether the modulation of posterior alpha power during working memory retention is partly driven by the eye movement patterns that different tasks naturally elicit. The study enrolled 120 participants, each completing 2 hours of simultaneous EEG and eye-tracking. We were able to demonstrate that posterior alpha power is modulated by task-dependent eye movement patterns. Furthermore, we demonstrated in a multiverse analysis that spectral parameterization can clean the EEG data from residual oculomotor artifacts.

A Multiverse Analysis on the Validity of Frontal Alpha Asymmetry as a Biomarker for Depressive Disorder in Children and Adolescents
Preregistered Open Data Open Materials Open Code
Hansen, A.D., Appenzeller, N., Goetz, M., Popov, T., Strzelczyk, D. & Langer, N.
[Code]
By systematically varying analytical decisions (electrode pairs, frequency bands, reference schemes, source models), this study tests how robust frontal alpha asymmetry is as an EEG biomarker for depression in a clinical child and adolescent sample, asking whether the effect survives choices that are theoretically arbitrary but frequently undisclosed.
